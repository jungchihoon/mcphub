- `server.performance` → `server.capabilities.performance`
- `ServerMetadata.security` → `compatibility.features` 활용

#### 2. 매칭 알고리즘 개선
- **임계값 조정**: 60 → 30 (테스트 환경)
- **시나리오 호환성**: mockServers와 일치하는 요구사항
